Un **chatbot**, c’est le format de conversation que vous voyez dans beaucoup d’outils d’IA.

Vous tapez une demande.
L’outil répond.
Puis vous continuez l’échange dans la même fenêtre.

## Ce que le mot désigne vraiment

Le mot **chatbot** parle surtout de la forme visible :

- une zone de discussion
- des messages qui s’enchaînent
- des réponses générées à la suite de votre demande

Autrement dit, le chatbot est la conversation.
Ce n’est pas forcément le moteur qui réfléchit derrière.

## Pourquoi c’est utile de faire la différence

Parce qu’on confond souvent :

- le mot large **IA**
- le **chatbot** qu’on a sous les yeux
- le **LLM** qui produit le texte

Pour se repérer simplement :

- **chatbot** = l’interface de discussion
- **LLM** = le moteur de langage
- **IA** = le terme plus large

## Ce qu’un débutant doit retenir

Le plus important n’est pas le vocabulaire parfait.
Le plus important est de savoir :

- ce que vous demandez
- ce que vous pouvez vérifier
- ce qu’il vaut mieux ne pas partager dans le chat

Le mot chatbot peut donc être pratique pour parler de l’outil, mais il ne dit pas tout à lui seul sur sa qualité ou sa fiabilité.
