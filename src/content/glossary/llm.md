**LLM** est un sigle anglais qu’on voit souvent passer dans les vidéos, les comparatifs ou les articles sur l’IA.

Pour un débutant, la version utile à retenir est simple :

> c’est le moteur qui produit ou transforme du texte quand vous écrivez une consigne.

## Pourquoi ce mot revient souvent

Parce que beaucoup d’outils populaires de conversation ou de rédaction reposent sur ce type de moteur.

C’est lui qui aide à :

- reformuler un message
- résumer un texte
- produire un premier brouillon
- répondre dans un chat

## Ce que ce mot ne veut pas dire

Le mot **LLM** ne veut pas dire :

- que l’outil comprend tout parfaitement
- que la réponse est forcément vraie
- que l’outil peut agir dans le monde réel tout seul

Un LLM peut très bien écrire avec assurance et se tromper quand même.

## Le bon repère

Si le mot vous intimide, retenez surtout ceci :

- **chatbot** = la forme de conversation
- **LLM** = le moteur de texte derrière
- **IA** = le mot plus large qu’on utilise pour parler de plusieurs types d’outils

Vous n’avez pas besoin de parler comme un spécialiste pour bien utiliser ces outils.
Vous avez surtout besoin de comprendre ce qu’ils font… et ce qu’ils ne font pas.
